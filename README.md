# aplikasi-database
Ini merupakan bahan latihan praktikum pembuatan sistem databases sederhana pada mata kuliah Databases pada Akademi Komunitas Mukomuko. 
- Php version = 7
- Database = Mysql
- Bootstrap v.3.7
